Washington LD KML Files

There is a KML file for each legislative district, named vt53_<n>.kml, where n is the LD number. Each file contains shapes for all precincts in the LD, including split precincts partially in the LD, colored with 11 different colors, ordered alphabetically. The overall LD shape with no fill is also included. Each precinct lists the total population and voter age population (VAP).

The KMLs work well in Google Earth (download at http://earth.google.com). They can be loaded in Google Maps, but because there are so many polygons, Google Maps will divide the polygons into 4 to 10 sections, so you cannot view them all at once.

All of these were produced by a tool I authored. Inputs are
- Precinct shapefiles (from Census 2010)
- Precinct population info (from Census 2010)
- Census block to precinct mapping info (from Census 2010)
- Block equivalency file (FINALLEG2011.txt) (from the state redistricting commission)
- LD District KML (FINAL_LEGISLATIVE_PLAN_2011.kmz) (from the state redistricting commission)
The tool also reduces the polygon complexity somewhat using the Douglas-Peucker algorithm.
